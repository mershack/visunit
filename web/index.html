VisUnit

A framework that supports web-based evaluations of data visualizations

VisUnit is free and open source. You can fork it on Github



Here are some useful links to functions of VisUnit

  • Designing user studies: VisUnit provides a user friendly interface that can be used to design user studies.

  • Managing user studies : VisUnit provides a user-friendly interface that can be used to manage existing user studies.

  • Creating Tasks: VisUnit provides a user-friendly interface that can be used to create tasks and survey questions.

  • Creating task instances : VisUnit provides a user-friendly interface that can be used to create task instances.

  • Adding Datasets: VisUnit provides a user-frienly interface that can be used to add more datasets.
