VisUnit

A framework that supports web-based evaluations of data visualizations

Installation instructions

  • VisUnit uses Java Servlets and Java on the backend, as such it will run on a webserver that supports Java such as Apache Tomcat Server. Download and install Apache Tomcat.
  • VisUnit uses "R" statistical package for the statistical analysis of the results. Download and Install the "R" statistical package.
  • Ensure "R" can be accessed from the terminal. For example if you type R on a Unix terminal or on a Windows command prompt , the version of R installed should be displayed.

    NB: Without R only the "Results analysis" functionality will be affected.
